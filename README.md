# scripts

这是一个个人脚本仓库，用来集中保存和迭代日常使用的小脚本、浏览器脚本、油猴脚本和自动化脚本。

## 当前脚本

- `chatgpt-conversation-tree.user.js`  
  ChatGPT 侧边栏对话分组油猴脚本：支持分组、拖动归类、未分组记忆、导入导出和菜单扩展。
- `本地文件处理脚本/一键GPT作品包/`  
  一键把 GPT 文案和同层图片整理成作品包：复制文案、下载图片、双击入口，即可生成 TXT、重命名图片、收进成品库，并在满 14 个作品包时自动整理作品集和 ZIP。

## 说明

这个仓库是私有仓库。当前脚本暂时保留在仓库根目录，是为了不破坏本地油猴安装和更新时使用的路径。后续如果脚本数量变多，再按类型整理到子目录。
