# 窗口布局启动器

这个工具用来保存并恢复一组工作窗口的位置和大小，例如：

- 左侧两个素材文件夹。
- 右侧 Edge / Chrome 里的 ChatGPT 窗口。

## 两个版本

### 应用版

路径：`应用版/release/窗口布局启动器.exe`

适合分享给别人。双击 exe 就是一个浅色玻璃拟态小应用，可以保存新布局、打开布局、覆盖布局、删除布局、导出分享包。

布局数据会存在 exe 同目录的 `layouts` 文件夹。别人第一次使用时，先摆好自己的窗口，再点“保存为新布局”即可。

如果要分享某一个布局，打开应用后选中布局，点击“导出分享包”。应用会在 `exports` 里生成 zip，里面包含 exe 和对应布局。

### 脚本版

路径：`脚本版/`

这是当前本机正在用的轻量脚本版，保留现有入口：

- `打开窗口布局中心.vbs`
- `保存为新窗口布局.vbs`
- `布局入口/打开-素材处理布局.vbs`

脚本版更轻，恢复已有窗口很快；应用版更适合给非技术同事开箱使用。

恢复布局时，如果对应窗口已经打开但最小化或在后台，会优先复用并拉回保存的位置，不会重复打开。

## 重新编译应用版

在 `应用版` 目录运行：

```powershell
.\build.ps1
```

编译结果会生成到：

```text
应用版\release\窗口布局启动器.exe
```
