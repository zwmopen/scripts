# 窗口布局启动器

这个工具用来保存并恢复一组工作窗口的位置和大小，例如：

- 左侧两个素材文件夹。
- 右侧 Edge / Chrome 里的 ChatGPT 窗口。

## 两个版本

### 应用版

路径：`应用版/release/窗口布局启动器.exe`

适合分享给别人。双击 exe 就是一个小窗口，可以保存新布局、打开布局、覆盖布局、删除布局。

布局数据会存在 exe 同目录的 `layouts` 文件夹。别人第一次使用时，先摆好自己的窗口，再点“保存为新布局”即可。

### 脚本版

路径：`脚本版/`

这是当前本机正在用的轻量脚本版，保留现有入口：

- `打开窗口布局中心.vbs`
- `保存为新窗口布局.vbs`
- `布局入口/打开-素材处理布局.vbs`

脚本版更轻，恢复已有窗口很快；应用版更适合给非技术同事开箱使用。

## 重新编译应用版

在 `应用版` 目录运行：

```powershell
.\build.ps1
```

编译结果会生成到：

```text
应用版\release\窗口布局启动器.exe
```
