# 窗口布局启动器 App

这是“窗口布局启动器”的应用版，目标是给别人直接拿去双击使用。界面采用浅色拟态悬浮风格：柔和灰蓝背景、浮起卡片、圆角按钮，并带有内置应用图标。

## 使用方法

1. 双击 `release\窗口布局启动器.exe`。
2. 把需要的文件夹、浏览器窗口摆好。
3. 点击“保存为新布局”，给布局起名。
4. 下次打开应用，选中布局后点击“打开布局”。

## 分享给别人

有两种方式：

### 分享整个应用

把 `release` 文件夹发给别人即可。对方双击：

```text
窗口布局启动器.exe
```

第一次使用时，对方摆好自己的窗口，然后保存为新布局。

### 分享某一个布局

在应用里选中布局，点击“导出 JSON”。应用会在 `exports` 文件夹生成一个布局 JSON 文件。

对方打开应用后点击“导入 JSON”，选择这个文件即可。

## 会保存什么

- 当前可见的文件资源管理器窗口路径、位置、大小。
- 当前标题包含 `ChatGPT` 的 Edge / Chrome 窗口位置、大小。
- 恢复布局时，如果对应窗口已经打开但最小化或在后台，会优先复用并拉回保存的位置，不会重复打开。

## 注意

- 这是绿色小应用，不需要安装。
- 布局数据保存在 exe 同目录的 `layouts` 文件夹。
- 如果换电脑、换显示器或改缩放比例，建议重新保存布局。
- 本地文件夹路径是绝对路径。别人电脑没有同样路径时，这个文件夹窗口不会自动打开，需要对方重新保存自己的布局。
- 应用只读取窗口标题、窗口位置、资源管理器路径，不读取网页内容。浏览器窗口主要按浏览器类型和标题关键字匹配，例如 `ChatGPT`。

## 界面方法论

- 风格名：拟态悬浮风。特征是冷灰蓝背景、柔和浮起卡片、圆角按钮、低饱和、蓝色主按钮。
- 本地小应用优先做“可用工具界面”，不要做营销页。
- 默认打开要居中，信息不能挤在角落，也不要留下大片无意义空白。
- 主内容区要能随窗口放大缩小；右侧操作按钮要保持可见并贴合右侧。
- 中文文字和自绘列表项要留足高度，避免标题和时间被裁切。
- 每次改 UI 后至少看三种状态：默认尺寸、放大、缩小。
