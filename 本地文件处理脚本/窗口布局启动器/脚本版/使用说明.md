# 窗口布局启动器

这个小工具用来保存并恢复“左边素材文件夹 + 右边两个 GPT 网页”的工作台布局。也可以保存多套不同工作布局。

## 平时怎么用

1. 先把两个文件夹、Edge 的 ChatGPT、Chrome 的 ChatGPT 摆到你喜欢的位置和大小。
2. 双击 `保存为新窗口布局.vbs`，给布局起个名字。
3. 下次开工时，双击 `打开窗口布局中心.vbs`，选择要打开的布局。

## 单独打开某一个布局

每次保存布局时，工具会自动在 `布局入口` 文件夹里生成一个入口：

```text
打开-布局名字.vbs
```

你可以直接双击它，启动对应工作台。

## 主要入口

- `打开窗口布局中心.vbs`：像一个小应用，可以选择布局、保存新布局、覆盖旧布局。
- `保存为新窗口布局.vbs`：直接保存当前窗口为一个新布局。
- `一键打开素材处理布局.vbs`：打开默认的“素材处理布局”。

## 它会保存什么

- 当前打开的文件资源管理器窗口路径、位置、大小。
- 当前标题里带 `ChatGPT` 的 Edge / Chrome 窗口位置和大小。

## 注意

- 浏览器恢复时默认打开 `https://chatgpt.com/`，如果原来的聊天页还开着，会优先摆放原窗口。
- 如果你换了显示器、缩放比例或分辨率，旧坐标可能会不合适，重新摆好后再点一次 `保存当前窗口布局.vbs` 就行。
- 这个工具只移动窗口，不会读取网页内容，也不会上传或处理素材。
